📊 Análisis de Evasión de Clientes (Churn) – TelecomX

📌 Descripción del Proyecto

Este proyecto tiene como objetivo analizar la evasión de clientes (Churn) en la empresa TelecomX mediante el uso de herramientas de análisis de datos en Python. A partir del análisis exploratorio de los datos, se busca identificar patrones y factores que influyen en la cancelación del servicio por parte de los clientes. Comprender las causas de la evasión es clave para que las empresas puedan diseñar estrategias que mejoren la retención de clientes y reduzcan la pérdida de ingresos.


🧹 Limpieza y Preparación de Datos

* Durante esta etapa se realizaron las siguientes actividades:

* Importación de los datos utilizando Python y Pandas.

* Normalización de columnas anidadas del archivo JSON.

* Verificación de valores nulos y registros duplicados.

* Estandarización de algunas variables categóricas.

* Creación de la variable Cuentas_Diarias a partir de los cargos mensuales.

Estas acciones permitieron obtener un dataset limpio y estructurado para el análisis.


📊 Análisis Exploratorio de Datos

* Se realizaron diferentes análisis y visualizaciones para comprender el comportamiento de los clientes, entre ellos:

* Distribución de la variable Churn para identificar la proporción de clientes que cancelaron el servicio.

* Análisis de variables categóricas como tipo de contrato, método de pago y servicio de internet.

* Comparación de variables numéricas como antigüedad del cliente (tenure) y cargos mensuales entre clientes que permanecen y clientes que cancelan el servicio.


🔎 Principales Hallazgos
El análisis permitió identificar algunos patrones importantes:

* Los clientes con contratos mensuales presentan mayor tendencia a cancelar el servicio.

* Los clientes con menor tiempo en la empresa tienen mayor probabilidad de evasión.

* Los cargos mensuales más altos pueden estar asociados con mayores tasas de cancelación.


💡 Recomendaciones
Con base en los resultados obtenidos, se sugieren las siguientes acciones:

* Promover contratos de mayor duración mediante incentivos o beneficios.

* Implementar estrategias de retención para clientes nuevos.

* Evaluar estrategias de valor en relación con los cargos mensuales del servicio.




Estas medidas pueden contribuir a mejorar la retención de clientes y reducir los niveles de evasión.
